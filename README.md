# Project Overview

This project provides a simple, fast way to interact with a local 3D scene through a web UI and a lightweight Node.js backend. The frontend offers an intuitive interface, and the backend exposes clean HTTP endpoints for scene actions and an AI-assisted chat to generate Python code for operations.

## Features
- Health and tools endpoints to inspect availability
- Chat API to generate Python for 3D actions
- Scene helpers: add cube/sphere, get scene info, execute code and a lot more 

## Tech Stack
- Frontend: React, Vite, TypeScript, Tailwind
- Backend: Node.js, Express, Axios

## Getting Started
- Backend: `cd backend && npm i && npm start`
- Frontend: `cd frontend && npm i && npm run dev`
